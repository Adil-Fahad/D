HALAL SCAN AI
An AI-powered cryptocurrency scanner that helps identify high-probability halal spot trading opportunities using machine learning, technical analysis, and institutional-grade risk filters.
Features
AI-powered trading signals
Halal cryptocurrency filtering
Binance market scanner
CoinDCX support
Machine learning predictions
RSI, EMA, MACD & Volume analysis
Institutional risk management
Watchlist and Telegram alerts
Tech Stack
Python
Flask
JavaScript
HTML/CSS
Machine Learning
Binance API
